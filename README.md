# Démarrer l’app React (Vite)

## Prérequis
- Node.js ≥ 18
- Un gestionnaire de paquets : npm, yarn ou pnpm

## Installation
# installe les dépendances
npm install
# ou
yarn
# ou
pnpm install

## Lancer l'app
npm run dev
# ou
yarn dev
# ou
pnpm dev


---

L’API est disponible par défaut sur :  
http://localhost:5173/
